## 🚀 部署就绪

### 环境要求
- ✅ Python 3.9+ 兼容
- ✅ Node.js 16+ 兼容
- ✅ MySQL 8.0+ 兼容
- ✅ 跨平台支持 (Windows/Linux/macOS)

### 快速启动
```bash
# 1. 初始化数据库
mysql -u root -p < backend/init_db.sql

# 2. 启动后端
cd backend
.\start.ps1  # Windows
./start.sh   # Linux/macOS

# 3. 启动前端
cd frontend
.\start.ps1  # Windows
npm run dev  # Linux/macOS

# 4. 访问应用
# 浏览器打开 http://localhost:5173
# 使用 admin/admin 登录
```
